üöÄ Getting Started

Welcome to Whisperrauth! This guide will help you set up and run the application locally.

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v20 or higher)
  • npm or yarn
  • Git
  • An Appwrite instance (cloud or self-hosted)

Installation

1. Clone the Repository

git clone https://github.com/whisperrnote/pass.git
cd pass

2. Install Dependencies

npm install
# or
yarn install

3. Configure Environment Variables

Copy the sample environment file and configure your Appwrite credentials:

cp env.sample .env.local

Edit .env.local and set the following variables:

# Appwrite Configuration
NEXT_PUBLIC_APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
NEXT_PUBLIC_APPWRITE_PROJECT_ID=your_project_id
NEXT_PUBLIC_APP_BASE_URL=http://localhost:3000

# Collection IDs (created in Appwrite)
NEXT_PUBLIC_USERS_COLLECTION_ID=your_users_collection_id
NEXT_PUBLIC_PASSWORDS_COLLECTION_ID=your_passwords_collection_id
NEXT_PUBLIC_DATABASE_ID=your_database_id
Note: You'll need to create collections in your Appwrite console. Refer to the Architecture guide for database schema details.

4. Run the Development Server

npm run dev
# or
yarn dev

Open http://localhost:3000 in your browser to see the application.

Setting Up Appwrite

1. Create a Project

  1. Go to your Appwrite console
  2. Create a new project
  3. Copy the Project ID to your .env.local

2. Create a Database

  1. Navigate to the Databases section
  2. Create a new database named "whisperrauth"
  3. Copy the Database ID to your .env.local

3. Create Collections

Create the following collections with appropriate permissions:

Users Collection

  • Name: users
  • Permissions: Document level
  • Attributes:
    • userId - String (required)
    • email - Email (required)
    • encryptedMasterKey - String
    • salt - String

Passwords Collection

  • Name: passwords
  • Permissions: Document level
  • Attributes:
    • userId - String (required)
    • encryptedData - String (required)
    • iv - String (required)
    • title - String
    • category - String
Security Note: All sensitive data is encrypted on the client-side before being sent to Appwrite. The server never has access to your unencrypted passwords.

First Login

  1. Navigate to http://localhost:3000
  2. Click "Register" to create a new account
  3. Enter your email and create a strong master password
  4. Your master password is used to encrypt all your data - never share it and don't lose it!
  5. After registration, you'll be logged in and can start adding passwords

Troubleshooting

Port Already in Use

If port 3000 is already in use, you can specify a different port:

npm run dev -- -p 3001

Appwrite Connection Issues

  • Verify your Appwrite endpoint is correct in .env.local
  • Check that your project ID and collection IDs are accurate
  • Ensure your Appwrite instance is running and accessible

Build Errors

  • Delete node_modules and package-lock.json
  • Run npm install again
  • Clear Next.js cache: rm -rf .next
